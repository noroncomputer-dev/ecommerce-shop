## ✨ ویژگی‌های پروژه

### 👤 بخش کاربری

- **Next.js 15** - فریم‌ورک React

- **Express** - فریم‌ورک وب
- **MongoDB** - دیتابیس
- **Mongoose** - ODM برای MongoDB
- **JWT** - احراز هویت
- **bcrypt** - هش کردن رمز
- **multer** - آپلود فایل

## ⚙️ نصب و راه‌اندازی

### پیش‌نیازها
- Node.js 18+
- MongoDB
- Git

### مراحل نصب

```bash
# 1. کلون کردن پروژه
git clone https://github.com/your-username/ecommerce-shop.git
cd ecommerce-shop

# 2. نصب وابستگی‌های بک‌اند
cd backend
npm install

# 3. تنظیم متغیرهای محیطی
cp .env.example .env
# ویرایش فایل .env با اطلاعات خودت

# 4. نصب وابستگی‌های فرانت‌اند
cd ../frontend
npm install
cp .env.local.example .env.local

# 5. اجرای پروژه
# ترمینال 1 - بک‌اند
cd backend
npm run dev

# ترمینال 2 - فرانت‌اند
cd frontend
npm run dev
